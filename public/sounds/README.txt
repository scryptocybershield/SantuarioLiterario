Sonidos para el Temporizador Pomodoro

El componente Pomodoro espera encontrar archivos de sonido en esta carpeta:

- rain.mp3: Sonido de lluvia suave (se reproduce al finalizar cada sesión)
- forest.mp3: Opcional (no se usa actualmente)

Actualmente los archivos están vacíos. Para una mejor experiencia, reemplázalos con archivos MP3 reales.

Recomendaciones:
- Usa sonidos relajantes sin copyright (puedes encontrar en https://freesound.org/)
- Duración: 10-30 segundos
- Volumen moderado (el componente ajusta volumen a 0.3)

Puedes usar herramientas online para convertir archivos de audio a MP3.
